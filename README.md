# Password Strength Checking and Hashing Utility

## Introduction

This is my project in the involvement of CS420 Introduction to Cyber Security Course. This project expresses the password strength checking 
and the utility of hashes used.

## Features

- Password Strength Checking
- Configuring Hash Parameters
- Hashing Utility
- Categorizing the password depending on their risk levels `Weak`, `Moderate`,`Strong` or `Critical`

## Requirements

This section is about which platforms or libraries have been used in this project.

- *JetBrains PyCharm (IDE used)*
- *Python*
- *Flask (For backend)*
- *Argon2id*
- *bcrypt*
- *HTML (For frontend)*

>[!IMPORTANT]
> The IDE used for Python is optional that you can also use another IDE to execute this project but when you take this project, you should be sure that
> you downloaded the required libraries and they properly work.
